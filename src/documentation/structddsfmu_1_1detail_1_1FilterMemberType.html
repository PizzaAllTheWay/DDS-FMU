Detailed Description

A content filter gets the DynamicPubSubType, which are instantiated as both fastdds and xtypes DynamicData. The fastdds DynamicData is first used to serialize the candidate sample, and the ddsfmu::Converter creates the xtypes DynamicData. This instance is used to compare key member values against those provided by the user.
